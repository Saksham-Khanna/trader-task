# Trader Behavior Insights

This project analyzes the relationship between trader performance and Bitcoin market sentiment
using historical trading data and the Fear & Greed Index.

The analysis explores profitability, win rate, and risk behavior across different
market sentiment conditions such as Fear and Greed.
